## Features

- **Dark / Light mode** toggle with localStorage persistence
- **Responsive design** — mobile, tablet, and desktop
- **Typed text animation** on the hero section
- **Smooth scroll** navigation with active highlight
- **Project categories** — filter by ML/DL, Web Dev, Game, App, IoT
- **Research section** with architecture thumbnails, DOI links, and full abstract modal
- **Transparent frosted-glass navbar** with mobile hamburger dropdown
- **Contact form** that opens a pre-filled mailto
